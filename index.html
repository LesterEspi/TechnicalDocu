What is Python

Python is an interpreted and object-orientedÂ high-level programming language with complex semantics. Its high-level built-in data structures, combined with dynamic typing and dynamic binding, make it ideal for Rapid Application Development as well as use as a scripting or glue language to link together existing components.

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

What can Python do?

  • Web Development - Python can be used on a server to create web applications
  • Software Development - Python can be used for rapid prototyping, or for production-ready software development.
  • Complex Mathematics - Python can be used perform complex mathematical calculations even on big data.
  • Data Mining - Python can be used to handle big data, connect to database systems, and read and modify files.

Python is becoming wildly used and as a matter of fact has become one of the best programming languages in 2021.

Why Python

Python's simple, easy-to-learn syntax emphasizes readability, which lowers software maintenance costs. Modules and packages are supported by Python, which facilitates software modularity and code reuse.

Advantages of using Python:

  • Python works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc).
  • Python has a simple syntax similar to the English language.
  • Python has syntax that allows developers to write programs with fewer lines than some other programming languages.
  • Python runs on an interpreter system, meaning that code can be executed as soon as it is written. This means that prototyping can be very quick.
  • Python can be treated in a procedural way, an object-oriented way or a functional way.

Compared to other programming languages, Python is generally the most user-friendly due to the use of new lines, indentation and whitespace to define scope and complete a command instead of using semicolons, parentheses, and curly brackets.

Start with the Basics

Python is an interpreted programming language, this means that as a developer you write Python (.py) files in a text editor and then put those files into the python interpreter to be executed.

Before anything else, let's write our first file, called helloWorld.py, in any text editor or any Python IDE.

print("Hello, world!")

If you have no Python Integrated Development Environment (IDE) Software such as PyCharm, IDLE or Eclipse, you can run your python file on the command line with this syntax:

C:\fileLocation>python fileName.py

For our example, the file name is helloWorld.py and is located in the Documents. Therefore, the code should look like this:

C:\Users\Documents>python helloWorld.py

After placing this on the command line, your output should look like this:

Hello, world!

You have now written and executed your first ever Python program!

Variables

As compared to some languages, Python has no command for declaring a variable. A variable is created when a value is initially assigned to it. For example:

a = 2600 b = "Baguio City" The variables a and b are created once 2600 and Baguio City have been assigned to them respectively.

Subsequently, because there is no formal declaration, the data types are not needed to be declared and Python will automatically detect the type of data you have assigned.

a = 35 a = "Maria" print(a)

When assigned the value of 5, the variable a became an integer. When Maria is assigned, variable a changed into a string type.

It should be noted that:

  • String variables can be declared by either using single or double quotes
  • Variable names are case sensitive
Data Types

In programming, data type is an important concept. Variables can store data of different types, and these different types can do different things.

In Python, there are built-in data types and are categorized into 7 categories:

  1. Text Type
    • str - string
  2. Numeric Types
    • int - integer(e.g. 1, 2, 3)
    • float - float(e.g. 2.1, 1.0, 3.1415)
    • complex - complex(e.g. 3 + 2x)
  3. Sequence Types
    • list - items are ordered and changeable
    • tuple - items are ordered and unchangeable
    • range - a sequence of numbers
  4. Mapping Type
    • dict - dictionary
  5. Set Types
    • set - items are mutable
    • frozenset - items are immutable
  6. Boolean Type
    • bool - boolean(true or false)
  7. Binary Types
    • bytes
    • bytearray
    • memoryview

To better understand this, here is a summary of all data types:

#str x = "apple" #int x = 5 #float x = 2.5 #complex x = 4x #list x = ["dog", "cat", "ball"] #tuple x = ("ball", "cat", "dog") #range x = range(2, 30, 2) #dict x = {"name" : "John", "age" : 36} #set x = {"ball", "cat", "dog"} #frozenset x = frozenset{"ball", "cat", "dog"} #bool x = True #bytes x = b"Hello" #bytearray x = bytearray(5) #memoryview x = memoryview(bytes(5))

You can also get the data type of a variable by using the type() function.

a = ["apple", "banana", "cherry"] b = "Baguio City" print(type(a)) print(type(b))

The result should look like the following:

<class 'list'> <class 'str'>

Type Casting

There may be times when you want to specify a type on to a variable. This can be done with casting. Python is an object-orientated language, and as such it uses classes to define data types, including its primitive types.

Casting in Python is therefore done using constructor functions

int() float() str()

In code it would look like

x = int(2.541) # x will be 2 y = float("1") # y will be 1.0 z = string(43) # z will be '43'
